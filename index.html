What Do Semantic-Elements Do?

Semantic-Elements clearly describe what is in that specific part of the web page.

Semantic Elements

Semantic Elements are the pieces of HTML code that clearly define each element of the web page. These elements define sections on the page such as the header, footer, articles, navigation (nav), time, and details. These elements will describe what information will be in each section. For example, the header details will be defined in the header element. The size, type, color, and other information will be included in order for the web browser to know how to display the header.
